Welcome to AnnotationBot’s documentation!

About the project

The project is a bot for creating and tracking schedules.

Installation

To install the project, you need to download or clone the repository from GitHub. Dependencies can be installed by running the command pip install -r requirements.txt from the root directory.

Usage

The project is used for tracking your schedule, as well as for querying currency exchange rates and weather information. There is also a function for adding new events to the schedule.
